About

Student Finance Tracker is a lightweight web app built to help students track day-to-day spending, set a monthly budget, and spot patterns in where their money actually goes. It supports multiple currencies, regex-powered search across transactions, and works fully offline using local storage — no account or server required.

Built with vanilla HTML, CSS and JavaScript as part of a Building Responsive UI assignment, focused on accessibility, mobile-first design, and clean modular code.
